# Cloud-Computing
Cloud Computing Project Analysing ATLAS Open Dataset - Based off HZZAnalysis example from: https://github.com/atlas-outreach-data-tools/notebooks-collection-opendata/blob/master/13-TeV-examples/uproot_python/HZZAnalysis.ipynb

Build images using:
```
./build.sh
```
**In the combiner-deployment.yaml change the local path to your full working directory or the directory you want your outputs to.**

Open Docker Desktop and Start Kubernetes. And use this bash script to run the deployment:
```
./run.sh
```
